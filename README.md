## Oh-my-zsh theme applying
Just replace the `zsh/prompt.zsh` to the `*.zsh-theme` file
#### Example: 
```bash
$ curl https://raw.githubusercontent.com/caiogondim/bullet-train.zsh/master/bullet-train.zsh-theme > zsh/prompt.zsh
```
